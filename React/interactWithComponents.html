Create forms in React

  • Two main ways to manage forms : controlled & uncontrolled
    • uncontrolled : the value of the input is not managed by the component
      • easier as there is less involvement from react
      • has its limitations
    • controlled : the value of the input is managed by the component
      • local state : holds specific component info, and comes from interaction between user and component
      • useState is a react function that changes values in the local state
      • good when checks need to be done on input
  • event.target['HTMLname'].value
    => to find out which element triggered the event
  • useState() (example in questionForms.js)
    • sets the initial value / state for inputValue,
    • indicates the corresponding function to modify it
    • set default value : ask your question here
