Pro. Devs. Don't Add React Using CDNs

They use automated tools to create a codebase containing essential tools already been pre-configured.

    Some functionalities :
  • Manage different dependencies (libraries) used by an app.
  • Optimize code fetching in browsers.
  • Import CSS and images.
  • Manage different versions of JavaScript.
  • Make developing easier by refreshing the page when the code is modified.

Create React App is a tool that will help you to do everything just mentioned.
There are other tools available (Next, Gatsby, Parcel, etc.), but Create React App remains the standard, especially for new React users.

allows you to generate a skeleton code for the app

includes some preconfigured tools such as Webpack, Babel, and ESLint
