Components

With React, each of these reusable parts is a component.

They contain everything they need to function correctly: structure, styles, and behavior.

It is every React developer's mission

to split each user interface into reusable elements, all nested within one another.
Most components are created by combining other, simpler ones.

HTML builds the DOM

React directly modifies the DOM

React.createElement()

React first creates elements in the virtual DOM and then warns the DOM when it is being created

JSX
is a JavaScript extension created by React

which means you can use syntax in the form of tags directly in the JavaScript code

the most easy-understandable way of writing React components in an application

so this method is used almost exclusively.
